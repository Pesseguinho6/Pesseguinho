Exercícios de CSS

1. Por suas palavras defina o CSS:
CSS é uma linguagem de estilo usada para definir a aparência visual de páginas HTML, como cores, fontes, espaçamento e layout.

2. Diga como é feita a ligação de uma página HTML com CSS externo:
Utilizando a tag <link> dentro do <head>, por exemplo:
<link rel="stylesheet" href="estilos.css">

3. Associe cada método de aplicação de CSS ao seu exemplo correspondente:
- Inline CSS: <p style="color: red;">Texto</p>
- Internal CSS: <style>p { color: red; }</style>
- External CSS: <link rel="stylesheet" href="estilos.css">

4. Regras no arquivo estilos.css:
- body com fundo cinza claro (#f4f4f4)
- h1 com fonte Arial, tamanho 32px, cor azul escura (#003087)
- p com texto justificado e margem de 15px

5. Problemas e soluções:
a) O erro de João foi usar um caminho absoluto, que só funciona no seu computador.
b) Deve usar um caminho relativo, como "css/estilizacao.css".
c) Estrutura:
projeto/
├── index.html
└── css/
    └── estilizacao.css
d) Linha correta:
<link rel="stylesheet" href="css/estilizacao.css">
